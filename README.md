# HotWheels 🏁 🏎️
## Introduction
Welcome to the main repository of the HotWheels team!  

This repo contains all the projects developed by Team01 (HotWheels) during the SEA:ME program in Portugal (2024-2025).  

## What is SEA:ME?
[SEA:ME](https://seame.space/) is a cutting-edge, tuition-free advanced studies program. In which students develop expertise in Code-driven Mobility, Autonomous Mobility, and Future Mobility Ecosystems. Over a 6-12 month period, students engage in expert-designed, practical coursework emphasizing hands-on, peer-to-peer learning in a lab environment. This immersive approach prepares them for work in the dynamic field of mobility IT.  

## Team's repository system
For the development of the projects during the program our team decided that the best way to organize would be through a system of submodules linked to a main repository. By using this method we were able to work on each submodule independently from the others, allowing features to be implemented or changed without interfering with other existing submodules. 
  
In this main repository is where where you can find all those submodules, each one of them representing a main project of the program.
