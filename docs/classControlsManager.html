ControlsManager::ControlsManager ( QObject * parent = nullptr)
explicit

Constructs a ControlsManager instance, initializing the engine controller, joystick controller, and setting up the necessary connections and threads.

This constructor sets up the EngineController signals to be connected to the ControlsManager signals. It initializes the JoysticksController with callbacks for steering and speed. It also initializes and starts the joystick controller in a separate thread.

Destroys the ControlsManager instance, ensuring that the joystick controller and its thread are properly cleaned up.

void ControlsManager::setMode ( DrivingMode mode)

Sets the current driving mode.

This function updates the m_currentMode member variable. If the mode has not changed, the function returns early. It is called to switch between Manual and Automatic modes.
